# 使用說明

## 安裝依賴（僅開發時需要）

```bash
pip install -r requirements.txt
```

## 運行程式（開發模式）

```bash
python main.py
```

## 打包為exe

### 方法1：使用build.spec（推薦）

```bash
pyinstaller build.spec
```

### 方法2：直接使用命令

```bash
pyinstaller --onefile --windowed --name=radioone main.py
```

打包後的exe檔案位於 `dist/radioone.exe`

## 使用方法

1. **啟動程式**
   - 雙擊 `radioone.exe` 運行
   - 程式會在同目錄下自動建立 `data` 資料夾用於保存播放計劃

2. **添加音訊檔案**
   - **方法1（推薦）**：拖放音訊檔案到視窗左側的拖放區域
   - **方法2**：點擊「選擇音訊檔案」按鈕，從檔案對話框選擇

3. **設定播放計劃**
   - 在右側「選擇周幾」區域勾選要播放的日期（週一到週日）
   - 設定播放時間（小時和分鐘）
   - 輸入計劃名稱（可選，預設為「上課提醒」）
   - 點擊「添加播放計劃」按鈕

4. **管理播放計劃**
   - 在播放計劃列表中可以看到所有已添加的計劃
   - **測試播放**：選中計劃後點擊「測試播放」按鈕
   - **編輯計劃**：雙擊計劃項或選中後點擊「編輯」按鈕
   - **刪除計劃**：選中計劃後點擊「刪除」按鈕

5. **後台運行**
   - 點擊視窗的「關閉」按鈕會將程式最小化到系統託盤
   - 右鍵點擊託盤圖示可以顯示視窗或退出程式
   - 播放時託盤圖示會閃爍提示

6. **查看狀態**
   - 視窗頂部顯示當前系統時間
   - 狀態列顯示播放狀態和下一個播放時間
   - 播放時會彈出Windows通知

## 注意事項

1. **音訊檔案路徑**
   - 程式會保存音訊檔案的完整路徑（絕對路徑）
   - 如果音訊檔案移動或刪除，程式會標記為失效但仍保留在列表中
   - 建議將音訊檔案放在固定位置或與程式放在同一目錄

2. **系統時間**
   - 程式依賴系統時間，請確保電腦系統時間準確
   - 視窗頂部會顯示當前系統時間供確認

3. **播放佇列**
   - 如果多個播放計劃在同一時間觸發，會自動排隊播放
   - 不會中斷當前正在播放的內容

4. **程式關閉**
   - 關閉視窗只會最小化到託盤，不會停止程式
   - 要完全退出程式，請右鍵點擊託盤圖示選擇「退出」
   - 所有播放計劃會自動保存

5. **便攜性**
   - 將exe檔案複製到隨身碟即可使用
   - 資料保存在exe同目錄的`data`資料夾中
   - 可以同時複製exe和音訊檔案到隨身碟，確保路徑一致

## 故障排除

1. **拖放功能不可用**
   - 如果顯示「tkinterdnd2未安裝」警告，請使用「選擇音訊檔案」按鈕代替

2. **音訊無法播放**
   - 檢查檔案格式是否支援（MP3, WAV, WMA等）
   - 檢查檔案路徑是否正確
   - 確認系統音量未靜音

3. **系統託盤不顯示**
   - 某些Windows版本可能需要額外的權限
   - 如果無法使用，可以保持視窗打開

4. **通知不顯示**
   - 檢查Windows通知設定是否開啟
   - 某些Windows 8系統可能需要啟用通知權限
