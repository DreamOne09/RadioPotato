# 更新說明

## 本次更新內容

### 1. ✅ 播放計劃列表顯示音訊檔案
- 新增「音訊檔案」列，顯示計劃中包含的音樂檔案名稱
- 如果檔案超過3個，會顯示前3個並提示總數（例如：檔案1.mp3、檔案2.mp3、檔案3.mp3... (共5個)）
- 列寬調整為220px，確保檔案名完整顯示

### 2. ✅ 放大版權資訊字體
- 版權資訊字體從9px增大到13px（粗體）
- 改為主要文字顏色，更清晰易讀
- 狀態列高度從50px增加到80px，容納更多內容

### 3. ✅ 修復重複顯示問題
- 隱藏Treeview的預設#0列（避免顯示兩行）
- 設定`show='headings'`只顯示自定義列
- 每行只顯示一次，不再重複

### 4. ✅ 確認排程器真實運行
- 程式啟動時會檢查並打印排程器狀態
- 排程器每秒檢查一次時間，匹配時自動觸發播放
- **確認功能**：程式確實會在指定時間播放，不是只有介面

### 5. ✅ 開機自動啟動功能
- 在底部版權資訊下方新增「開機時自動啟動」選項
- 勾選後自動添加到Windows開機啟動項
- 取消勾選會從啟動項移除
- 使用Windows註冊表實現，無需管理員權限

## 功能確認

### 排程器運行機制
排程器使用獨立執行緒每秒檢查一次當前時間：
1. 比較當前時間（HH:MM）與播放計劃的時間
2. 比較當前周幾與播放計劃的周幾
3. 如果匹配且今天未觸發過，自動觸發播放
4. 觸發後加入播放佇列，按順序播放

**確認**：這是真實的自動播放功能，不只是介面展示。

## 使用開機自動啟動

1. 勾選底部的「開機時自動啟動」選項
2. 顯示「已設置開機自動啟動」訊息
3. 重新開機後程式會自動啟動
4. 可以取消勾選來移除自動啟動

## 播放計劃列表說明

現在列表中會顯示：
- **名稱**：播放計劃名稱
- **週幾**：播放的日期（週一、週二等）
- **時間**：播放時間（HH:MM格式）
- **音訊檔案**：音樂檔案名稱列表（前3個+總數）
- **檔案數**：總共有多少個音訊檔案

這樣可以清楚看到每個計劃包含哪些音樂！
